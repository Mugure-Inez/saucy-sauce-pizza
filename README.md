# Project name

Saucy Sauce Pizza is reopening again better and refined, this webpage should help the users quickyl choose and see the pizza options available at the store.

## Author

Mugure Inez

## Project description

This new and improved webpage is easy to use. From the landing page users are redirected to the pickup page to place an order and later to the checkout page where they can confirm delivery information.

## Technologies used

- Javascript used to make the page interactive
- CSS used to style the page
- HTML used to structure the outlook of the page

## Setup & Installation instructions

- Project can be forked and cloned on github [https://github.com/Mugure-Inez/saucy-sauce-pizza]
